Proyectos

WebApp "Glucky"

Este fue el primer proyecto que desarrollé con mi equipo en el CECyT 9, el cual constaba de una aplicación Web para mantener un control sobre los niveles de glucosa en la sangre para aquellas personas que padecen de diabetes, de esta manera se ahorraba tiempo y recursos para tanto los pacientes como para los endocrinólogos ya que los registros se guardaban continuamente y había un chat de comunicación para cada paciente con su respectivo doctor.

WebApp de IA "Lyra"

Este proyecto se trataba sobre una aplicación Web y el uso de una API de openai, la cual fue entrenada para tomar la función de un psicólogo, no completamente ya que la idea principal era ayudar a las personas timidas y/o con depresión a externar sus emociones por medio de voz, usando text-to-speach, generando un resumen sobre la platica del paciente al psicólogo y de esta manera eficientar el diagnóstico.

WebApp "Sage Biomedical"

Por último, el proyecto mas reciente que estoy desarrollando como freelancer es esta aplicación web para la empresa Sage Biomedical, la cual es una empresa de venta y servicios a equipos médicos. Se divide en dos secciones, la parte donde cualquier persona que acceda a la página verá la información de contacto, productos en venta, posibles servicios, etc. Y la segunda, donde se maneja la logística interna de la empresa.
